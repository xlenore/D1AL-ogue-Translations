# D1AL-ogue Translations

## Project Structure

- The `translations` folder contains the translation texts:
	- `human`: Translations made by humans.
	- `ia`: Machine-generated translations (AI).  
		**Note:** AI translations are placeholders only and will never replace human work. Anyone is welcome to contribute human translations for any language.

- The `patch` folder contains the ready-to-use, compiled files for the game.

## How to Install the Translation Patch

1. Locate your game installation folder, typically at:
	 ```
	 \steamapps\common\D1AL-ogue\D1AL-ogue_Data\StreamingAssets\aa
	 ```

2. Replace the following files with the ones from the `patch` folder:
	 - Replace `catalog.bin` in:
		 ```
		 \steamapps\common\D1AL-ogue\D1AL-ogue_Data\StreamingAssets\aa
		 ```
	 - Replace `localization-string-tables-english(unitedstates)(en-us)_assets_all.bundle` in:
		 ```
		 \steamapps\common\D1AL-ogue\D1AL-ogue_Data\StreamingAssets\aa\StandaloneWindows64
		 ```

## Contributing
